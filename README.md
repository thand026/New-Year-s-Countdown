# New-Year-s-Countdown
A few Lines of Code that countdown to 0

The programs ask the user for an input as to where to start the countdown and through system threading sleep, the program outputs number by number all the seconds down until the countdown reaches zero

# How To Run
You will need a python or C# interpreter

# Python
Open a terminal and navigate to the countdown.py folder
run the countdown.py

# C#
Run the Program.cs file using a C# interpreter
